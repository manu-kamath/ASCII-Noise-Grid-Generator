# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0]
### Changed
- Updated the version number displayed in the application header to "v1.3".

## [1.2.0]
### Added
- SVG export options for both "Landscape" and "Portrait" orientations, mirroring the PNG export functionality.

### Changed
- The "Export Options" section has been restyled to be visually consistent with other UI panels in the settings drawer.

### Removed
- The "Android Guide" and "iOS Guide" buttons and their associated help text have been removed to simplify the UI.

## [1.1.0]
### Fixed
- Corrected a layout issue with the "Shape Gradient" toggle switch. The label and switch are now properly aligned on a single line, preventing the label text from wrapping.

## [1.0.0]
### Added
- Initial release of the ASCII Noise Grid Generator.
- Core features include:
  - Real-time animated ASCII noise canvas.
  - Selection of default shapes and custom SVG uploads.
  - Display modes: "Center" and "Tile".
  - Controls for animation speed, character sets, and opacity.
  - Full color customization with individual color pickers and preset palettes.
  - Export options for PNG (landscape/portrait) and a self-contained HTML file.
